# 🌍 Air Quality & Weather Monitoring Web App

This is a responsive and animated web application built using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js) that allows users to search for any city and get real-time **air pollution (AQI)** and **weather** data using satellite and ground-based observations.

---

## 🔧 Tech Stack

- **Frontend:** React.js + MUI (Material UI), Framer Motion, React Leaflet
- **Backend:** Node.js + Express.js
- **Database:** MongoDB Atlas (if storing data)
- **APIs used:**
  - [OpenWeatherMap Air Pollution API](https://openweathermap.org/api/air-pollution)
  - [OpenWeatherMap Weather API](https://openweathermap.org/current)
  - Optionally: OpenAQ, CAMS, MERRA-2, Sentinel-5P via GEE

---

## ✨ Features

- 🌫️ Real-time Air Quality Index (AQI) with pollutant breakdown (PM2.5, PM10, NO₂, etc.)
- 🌤 Live weather conditions (temperature, humidity, wind speed)
- 🗺️ Interactive map (Leaflet) showing the selected location
- 💡 Animated and modern UI using MUI + Framer Motion
- 📱 Mobile responsive design
- 🌈 Bubble animation in hero section for attractive feel
